# Magic Cube

魔方是新一代的资源代理工具，使用最灵活的规则来作为基础，可以适应不同的代理需求。

## 安装

windows:
```
npm install -g httpx
```
linux/mac:
```bash
$ sudo npm install -g httpx
```

## 使用
1. 在命令行输入mc，默认端口为80，可以使用```mc -p xxxx```来修改端口
2. 打开 ```http://127.0.0.1:xxxx```访问首页进行配置
3. enjoy it

## 参数

 * -p [number] 端口号
 * -v 当前版本
